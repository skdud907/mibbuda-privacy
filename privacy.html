미쁘다 개인정보처리방침

최종 수정일: 2026-07-08

미쁘다(이하 "앱")는 여행 회비와 지출 기록을 사용자가 직접 관리할 수 있도록 돕는 여행 총무 가계부 앱입니다. 본 방침은 앱이 어떤 정보를 어떻게 처리하는지 안내합니다.

1. 수집하는 정보

앱은 아래 정보를 사용자가 직접 입력하거나 기기 권한을 허용했을 때 처리합니다.

  • 여행명, 국가, 지역, 참여자 이름 등 사용자가 입력한 여행 정보
  • 회비, 추가 회비, 지출 금액, 결제 수단, 메모 등 사용자가 입력한 기록 정보
  • 영수증 및 방문 사진 이미지
  • 위치 권한 허용 시 현재 위치 정보
  • 항공, 숙소, 외부 링크 등 사용자가 기록한 참고 정보

앱은 이름·전화번호·이메일 등 회원가입 기반 개인정보를 별도로 수집하지 않습니다.

2. 정보 이용 목적

  • 여행 회비와 지출 정리
  • 남은 공금 및 개인 선결제 정산 계산
  • 영수증과 방문 기록 첨부
  • 사용자가 저장한 여행 정보 표시

3. 저장 및 공유 방식

  • 입력한 데이터와 첨부 이미지는 기본적으로 기기 내부 로컬 저장소에 저장됩니다.
  • 사용자가 초대 코드로 여행을 공유하는 경우, 해당 여행의 데이터는 함께 참여한 일행이 볼 수 있도록 클라우드(보안 데이터베이스)를 통해 동기화됩니다. 이 동기화는 사용자가 공유를 선택했을 때만 이루어집니다.
  • 별도의 회원 계정 가입은 요구하지 않습니다.

4. 권한 사용 안내

  • 카메라: 영수증 또는 방문 사진을 촬영해 첨부하기 위해 사용합니다.
  • 사진 라이브러리: 기존 이미지를 첨부하기 위해 사용합니다.
  • 위치: 지출 또는 방문 기록에 현재 위치를 자동 입력하기 위해 사용합니다.

모든 권한은 사용자가 허용한 경우에만, 해당 기능을 사용할 때만 쓰입니다. 거부해도 나머지 기능은 정상 이용할 수 있습니다.

5. 제3자 제공

앱은 사용자가 초대 코드 공유 등으로 직접 공유하지 않는 한, 입력 정보를 외부 제3자에게 전송하거나 판매하지 않습니다. 환율·국가 정보 조회를 위해 외부 공개 API를 호출할 수 있으나, 이때 사용자의 개인 식별 정보는 전송되지 않습니다.

6. 보관 및 삭제

  • 사용자가 앱 내부에서 기록을 삭제하면 해당 데이터는 앱 저장소에서 삭제됩니다.
  • 앱을 삭제하면 기기에 저장된 로컬 데이터와 첨부 파일도 함께 제거될 수 있습니다.
  • 공유된 여행 데이터의 삭제를 원하시면 아래 문의처로 요청해 주세요.
